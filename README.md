**Smartphone Market Analysis (Python – EDA)**

Overview  :
This project analyzes smartphone consumer preferences using Python.  
The goal is to understand customer demographics, purchasing behavior, and brand preferences.

---

Tools & Libraries : 
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

Key Steps : 

1. Data Cleaning
- Handled missing values using mode imputation
- Standardized categorical values
- Prepared dataset for analysis

2. Exploratory Data Analysis (EDA)
- Univariate analysis (distribution of demographics and usage patterns)
- Bivariate analysis (relationship between income, age, brand, etc.)
- Multivariate analysis for deeper insights

---

Key Insights  : 
- Majority users belong to the **18–25 age group**
- Most users prefer smartphones in the **₹10,000–₹20,000 range**
- **Offline retail stores** are the most common purchase channel
- Users show **price sensitivity and feature-driven preferences**

---

Dataset  : 
Survey-based dataset containing information on:
- Demographics
- Income
- Smartphone usage
- Brand preferences
- Purchase behavior
